# Investment_game
骰子游戏记录表



这个骰子游戏记录表统计程序具备以下特点：


## 用户权限管理：

管理员：可添加、查看、删除记录，可查看图片和视频

普通用户：只能查看任务名称和基本信息

## 数据记录功能：

记录任务发布时间（显示年月日星期几）

任务名称支持多行文本

支持上传图片或视频，并提供预览功能

记录任务完成时间

标记是否触发曝光

## 界面设计：

响应式布局，适配各种设备

美观的卡片式展示

平滑的动画和过渡效果

模态框设计，不遮挡主界面

## 其他功能：

图片 / 视频支持全屏查看

底部有下载和分享按钮

数据存储在 data.json 文件中

默认管理员账号：admin

默认密码：admin123

你可以直接使用这个单文件程序，它会自动创建所需的目录和文件。
